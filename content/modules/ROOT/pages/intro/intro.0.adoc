== Workshop Overview

Welcome to the *Build fast and more secure AI data flows with Red Hat Application Foundations and Red Hat Connectivity Link* workshop! This workshop is designed to provide you with a comprehensive introduction to building AI data flows using Red Hat's powerful tools and technologies. Whether you're new to AI or an experienced developer, this workshop will guide you through the process of creating efficient and secure data flows that can power your AI applications.

{empty} +

=== Workshop Progression

For the best experience, we recommend following the workshop in order. Each section builds on the previous one, allowing you to gradually develop your skills and understanding of the concepts, but feel free to select specific modules that interest you the most. In the Home page, you'll find a roadmap that outlines the progression of the workshop, along with links to each module. You can also use the navigation menu to jump directly to any section.
