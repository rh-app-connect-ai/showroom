. *Personalise the Seller information*
+
In this invoice, *you* are the Seller. The Seller fields currently pull values from the EDI source using XPath expressions — let's replace them with your own location. Overwrite the following fields in the *Target* with hardcoded values:
+
--
... `Name` — your company or building name, e.g. `"My Company Inc"` +
... `Street` — street address +
... `City` +
... `State` +
... `PostalCode`
--
+
IMPORTANT: String values must be wrapped in double quotes (e.g. `"Atlanta"`). Without quotes, the data mapper treats the value as an XPath expression instead of a literal string.
