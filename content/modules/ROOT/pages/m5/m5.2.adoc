+
[.copypaste]
****
step 22

=== Test Scenarios

Test the following scenarios by updating the API key value and sending a message through the chat interface:

{blank}

. **Invalid credentials** -- Set an invalid API key:
+
[.copypaste]
****
forage.myagent.agent.api.key=invalidkey
****
+
Send a message. The request should be **denied** (401 Unauthorized).
+
{empty} +

. **Valid free-tier key** -- Set the free-tier API key:
+
[.copypaste]
****
forage.myagent.agent.api.key=iamafreeuser
****
+
Send a message. The request should be **allowed** -- you receive a response from the LLM.
