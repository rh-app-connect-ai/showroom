=== Start the flows

Scale up all four deployments to start the flows:

{blank}

[.copypaste]
****
oc scale deployment m2k r2k k2m k2r --replicas=1
****

[TIP]
--
.What does this command do?
[%collapsible]
====
* `oc scale deployment` — adjusts the number of running replicas for one or more deployments
* `m2k r2k k2m k2r` — the four Camel integration flows
* `--replicas=1` — scales each deployment to one replica, starting them up
====
--

Once all pods are up and running, the Topology view should show blue rings around each deployment:

[.bordered]
image::m3/openshift-topology-running.png[width=60%]

=== Test end-to-end

Now that all flows are running, it's time to simulate a conversation between customer and support agent. Make sure both platforms are open:

* xref:m2/m2.1.adoc[Matrix platform onboarding] — the customer side
* xref:m2/m2.2.adoc[Rocket.Chat platform onboarding] — the support agent side

{empty} +

Let's simulate a brief exchange between a customer and a support agent.

. *Customer sends a message* — go to your _Matrix_ room and type:
+
{blank}
+
[.copypaste]
****
Hi, I need some help with a purchase order I placed recently.
****
+
{blank}

. *Agent responds* — go to your _Rocket.Chat_ room, you should see the customer's message. Reply with:
+
{blank}
+
[.copypaste]
****
Of course! Could you share your order reference so we can look into it?
****

{empty} +
