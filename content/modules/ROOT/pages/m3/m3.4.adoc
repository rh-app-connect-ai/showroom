== Enable all flows

You've built and deployed the _Kafka_ to _Rocket.Chat_ flow from scratch. The remaining flows follow the same pattern — to save time, let's enable them all at once and test the full end-to-end integration.

{empty} +

[.mermaid]
----
flowchart LR
    M([Matrix]) --> K([Kafka])
    K --> RC([Rocket.Chat])
    RC --> K
    K --> M

    style M fill:#e8f5e9,stroke:#66bb6a
    style K fill:#fff3e0,stroke:#fb8c00
    style RC fill:#e8f5e9,stroke:#66bb6a
----

{empty} +

=== Deploy the remaining flows

Run the following command to deploy all remaining flows on _OpenShift_:

{blank}

[.copypaste]
****
labdeploy all
****

{blank}

NOTE: Be patient, this action may take some time to complete.

{empty} +

=== Test end-to-end

Now that all flows are running, send a message from _Matrix_ and watch it arrive in _Rocket.Chat_ — routed through _Kafka_.

{empty} +

. Go to your _Matrix_ room and send a message, for example `Hello from Matrix`.

. Open your _Rocket.Chat_ room — the message should appear:
