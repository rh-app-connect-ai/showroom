= Local Testing

== Showroom (documentation)

Runs the Antora documentation site with live reload on http://localhost:8080.

Uses `local-site.yml` which sets `username: user11` for local testing. To change the test username, edit the `username` attribute in `local-site.yml`.

[source,bash]
----
podman run --rm --name antora -v $PWD:/antora -p 8080:8080 \
  -e ANTORA_CONFIG=local-site.yml \
  -i -t ghcr.io/juliaaano/antora-viewer
----

== DocServer (credentials API)

The DocServer provides user credentials to copypaste blocks at runtime. A local Camel-based mock is included with hardcoded test data.

Requires https://www.jbang.dev[JBang] with the Camel CLI (`camel`) installed:

[source,bash]
----
jbang app install camel@apache/camel
----

Start the DocServer:

[source,bash]
----
./camel/docserver/local/run.sh
----

The DocServer runs on port 9090, matching the `:url-docserver:` attribute defined in `content/modules/ROOT/partials/vars/urls.adoc`.

Stop the DocServer:

[source,bash]
----
camel stop docs
----

Or stop all running Camel integrations:

[source,bash]
----
camel stop
----

== Running both together

Start the DocServer in the background, then run Showroom:

[source,bash]
----
./camel/docserver/local/run.sh

podman run --rm --name antora -v $PWD:/antora -p 8080:8080 \
  -e ANTORA_CONFIG=local-site.yml \
  -i -t ghcr.io/juliaaano/antora-viewer
----

The documentation will show a green status dot when the DocServer is reachable, and copypaste blocks with `[.path]` will fetch live data.

== How it works

....
┌──────────────────────┐       ┌──────────────────────┐
│  Showroom (Antora)   │       │  DocServer (Camel)   │
│  localhost:8080      │       │  localhost:9090       │
│                      │       │                      │
│  Renders docs with   │ fetch │  Returns per-user    │
│  copypaste blocks   ─│──────>│  credentials using   │
│  JS sends "user"     │       │  the "user" header   │
│  header from cookie  │<──────│                      │
└──────────────────────┘       └──────────────────────┘
....
